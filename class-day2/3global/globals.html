2. let x = 5 — NOT on global  |  window.y = 55 — ON global

let x = 5  → window.x not global
window.y = 55  → window.y global
var z = 99  → window.z var leaks to global!
Rule: let / const are block-scoped — never go on window.
var at top-level DOES leak to window (avoid this).
window.y = 55 explicitly puts y on the global object — accessible everywhere.
